void bot_drive_distance ( Behaviour_t *  caller,
int8_t  curve,
int16_t  speed,
int16_t  cm 
)

Das Verhalten laesst den Bot eine vorher festgelegte Strecke fahren. Dabei legt die Geschwindigkeit fest, ob der Bot vorwaerts oder rueckwaerts fahren soll.

Parameter:
*callerDer Aufrufer
curveGibt an, ob der Bot eine Kurve fahren soll. Werte von -127 (So scharf wie moeglich links) ueber 0 (gerade aus) bis 127 (so scharf wie moeglich rechts)
speedGibt an, wie schnell der Bot fahren soll. Negative Werte lassen den Bot rueckwaerts fahren.
cmGibt an, wie weit der Bot fahren soll. In cm :-) Die Strecke muss positiv sein, die Fahrtrichtung wird ueber speed geregelt.

Definiert in Zeile 96 der Datei behaviour_drive_distance.c.

Benutzt ENCODER_MARKS, WHEEL_PERIMETER, sensEncL, sensEncR, switch_to_behaviour(), bot_drive_distance_behaviour() und BEHAVIOUR_NOOVERRIDE.

Wird benutzt von bot_unload_pillar_behaviour(), bot_drive_area_behaviour(), bot_drive_square_behaviour(), border_follow_wall_handler(), hang_on_handler(), bot_explore_behaviour(), bot_do_slalom_behaviour(), bot_simple_behaviour(), bot_solve_maze_behaviour() und bot_transport_pillar_behaviour().
