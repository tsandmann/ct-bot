Ausführliche Beschreibung

Kalibriert die Distanzsensoren des Bots.

Einige Zeilen sind auskommentiert. Sie waren dazu gedacht, die Kalibrierung zu automatisieren, das geht aber zurzeit (noch) nicht, weil die Positionsbestimmung fuer kleine Distanzen etwas zu ungenau ist (=> Encoderdatenauswertung checken!)
