Ausführliche Beschreibung

Deaktiviert ein anderes Verhalten in Abhaengigkeit einer Check-Funktion.

So kann z.B. der Wandfolger (bot_solve_maze) beendet werden, falls dieser auf eine Linie faehrt und der Linienfolger uebernehmen.
