Startet eine neue Keypad-Eingabe.

Abgeschlossen wird sie mit der Taste "Play", abgebrochen mit "Stop". Nach Abschluss wird die uebergebene Callback-Funktion aufgerufen mit dem Eingabe-String als Parameter.

Parameter:
*callbackZeiger auf eine Funktion, die die Eingabe bekommt
modevoreingestellter Modus (0: alpha-num, 1: numerisch)
rowZeile der Cursorposition fuer die Anzeige der Eingabe
colSpalte der Cursorposition fuer die Anzeige der Eingabe

Definiert in Zeile 160 der Datei gui.c.

Wird benutzt von remotecall_display().
